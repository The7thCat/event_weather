Work in Progress!

The goal is to provide weather data for your local running events eg. Tallinn marathon.

Currently you get a forecast for up to 9 days. The first couple of days give you hourly temperature data.
After that it switches to whatever data is provided by the source.
Currently the forecast uses the MET Norway data. 
For each day of the forecast a seperate graph will be plottet.

Next thing to do:
If the event is in a couple of weeks/month the historical button should get the data from past years
to get an idea what you might face. Currently this implementation will use metostat. But it's under development
and not yet implemented.

When this works as inteded other data will be added like wind, humidity, rain, etc.
